## Welcome to SideCar

SideCar is a **local-first** AI coding assistant for VS Code. It runs on your own machine by default — point it at [Ollama](https://ollama.com) and nothing leaves your laptop. Plug in an Anthropic or OpenAI key when you want frontier-model reasoning, and SideCar will automatically delegate heavy research to a local worker to keep your spend down.

**What you get out of the box:**

- Chat in a sidebar panel (`⌘⇧I` / `Ctrl+Shift+I`)
- Inline chat in the editor (`⌘I` / `Ctrl+I`)
- Code actions on diagnostics (the 💡 lightbulb)
- Agent mode with file edits, shell commands, and git operations
- Session cost tracking for paid backends
- Pending-change review with native file decorations

This walkthrough covers the essentials. Each step takes under a minute — you can come back to it any time from the Command Palette (`SideCar: Open Walkthrough`).
