Getting Started

~5 min read · Installation & first run

Install Ollama

Download and install Ollama for your platform. After installation, verify it's working:

shell 
ollama --version

Pull a recommended model:

shell 
ollama pull qwen3-coder:30b
Tip — lower RAM?

On machines with less RAM, try qwen3-coder:8b or qwen2.5-coder:7b instead.

Install SideCar

Install from the VS Code Marketplace, or search for SideCar in the VS Code Extensions panel (Cmd+Shift+X / Ctrl+Shift+X).

Your first chat

  1. Click the SideCar icon in the activity bar (left sidebar)
  2. Type a message in the chat input
  3. SideCar auto-launches Ollama if it's not already running

SideCar automatically includes your active file and workspace context in every conversation.

Choosing a model

Use the model dropdown at the top of the chat panel to browse and switch models. Models are organized into two categories:

  • Full Features (Tools) — models that support function calling (e.g. qwen3-coder, llama3.1, command-r). Required for agentic mode.
  • Chat-Only — models that work for conversation but can't use tools (e.g. gemma2, llama2, mistral).

You can also install new models directly from the dropdown — SideCar pulls them from the Ollama registry automatically.

Using the Anthropic API

To use Claude models instead of local Ollama:

settings.json 
{
  "sidecar.baseUrl": "https://api.anthropic.com",
  "sidecar.apiKey": "sk-ant-...",
  "sidecar.model":  "claude-sonnet-4-6"
}
Prompt caching

SideCar uses prompt caching with the Anthropic API, reducing input token costs by ~90% on cache hits.

OpenAI-compatible servers

SideCar works with any server that exposes the OpenAI /v1/chat/completions endpoint — including LM Studio, vLLM, llama.cpp, text-generation-webui, and OpenRouter.

Claude Code skills

SideCar can load and use Claude Code skills directly — no format conversion needed. Skills are loaded from:

  • ~/.claude/commands/ — your user-level Claude Code skills
  • .claude/commands/ — project-level Claude Code skills
  • .sidecar/skills/ — SideCar native skills

Trigger a skill with /skill-name in chat, or SideCar will match skills automatically based on your message. Type /skills to see all available skills.
